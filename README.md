# Airbnb Pricing Intelligence & Revenue Optimization Analysis

## Overview

This project analyzes the New York City Airbnb market from the perspective of a **Property Management Team**. The objective is to understand pricing patterns, customer demand, and market opportunities while developing a machine learning model to predict Airbnb listing prices.

## Business Questions

- Which boroughs command premium pricing?
- Which neighbourhoods offer the best investment opportunities?
- How does room type influence pricing and demand?
- Which factors have the greatest impact on listing prices?
- Can Airbnb listing prices be predicted using machine learning?

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## Key Findings

- Manhattan emerged as the strongest premium market.
- Chelsea, Financial District, Midtown, and Theater District showed strong investment potential.
- Private Rooms generated the highest customer engagement, while Entire Homes achieved the highest prices.
- XGBoost delivered the best predictive performance (CV R² = 0.63, Test R² = 0.65).
